MERN Stack Website

Electro Mart

- Introduction:

The ElectroMart project aims to develop an ecommerce website focusing on technology products, targeting a diverse range of users from young people, teenagers to the elderly, to provide convenience through online shopping and payment services.
In this project, there is interaction between two main entities: admin and user, to control and manage products and orders.

- Point: 8.5

- The technologies used:

+ MongoDB

+ Express.js

+ Node.js

+ React.js

- Number of participants: 3 - Position: Leader

- Roles:

Led end-to-end project execution, encompassing planning, coding, design, and interface functionality development. Created a list of functions and mockups, employed MongoDB for data storage, crafted backend APIs with Express.js, and built frontend features using React.js.

- Function List

Admin:

+ Login (If the information is correct, save the token and update the global state according to the username).

+ Homepage: display a welcome page with a GIF image.

+ Logout (Delete the token and set user information to default).

+ Manage categories (By ID and name).

+ Manage products (By ID, Name, Price, Creation Date, Category, and Image).

+ Manage orders, including Order ID, Creation Date, Customer Name, Total Amount, Status pending or approved for pending status, can be Approved or Canceled.

+ Manage customers, including ID, username, password, name, phone number, email, and activation status Email (send activation email containing ID and Token to customers) or Deactivate (deactivate customer account).

Customer:

+ Register (Including username, password, Name, Phone, Email, save token after registration, customers will receive an activation notification via email).

+ Login (If the information is correct, save the token and update the global state according to the username with the condition that the account is active).

+ Activate account (Customers enter ID and token - obtained from mail after the admin selects the Email option to send activation mail to the customer).

+ Update profile (Username, password, Name, Phone, Email already filled in).

+ Homepage (Including banner, product categories, customer reviews, footer).

+ Statistic of the number of products added to the shopping cart (based on items excluding quantity of the same product type).

+ Manage shopping cart (Including ID, Name, Category, Image, Price, Quantity, Amount (total), Action).

+ Checkout (Payment for products after adding them to the shopping cart, send order request to admin, status at this time is pending).

+ View product categories.

+ View product details (Including ID, Name, Price, Category, Quantity).

+ View order status.

+ Google Map API.

+ Product search.

+ Live chat.

+ Dark mode.
